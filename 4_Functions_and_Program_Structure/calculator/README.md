# The Ultimate RPN Terminal Calculator 🚀

Welcome to the **Ultimate Reverse Polish Notation (RPN) Calculator**! This lightweight, terminal-based calculator is designed for speed, precision, and raw calculating power.

## Features
* **Reverse Polish Notation:** Evaluate complex expressions cleanly without ever needing parentheses! (e.g., `2 3 +` instead of `2 + 3`).
* **Robust Parsing:** Effortlessly handles integers, decimals, and negative numbers.
* **Variable Storage:** Use lowercase letters to store and recall your calculations on the fly.
* **Streamlined Engine:** Powered by a highly optimized, self-contained lexical analyzer using a one-character memory state (zero external buffering required!).

## How to Use
Launch the executable and type your expressions in RPN format, separating numbers and operators with spaces. Press `Enter` to evaluate and print the result.

### Examples
```text
> 4 5 +
9

> 10 2 /
5

> -5 2.5 *
-12.5

> 5 1 2 + 4 * + 3 -
14
```

### Supported Operations
* `+` : Addition
* `-` : Subtraction
* `*` : Multiplication
* `/` : Division
* Math operators and variables functionality built in!
